语句
del 语句
	1、语法：
		del 变量名1，变量名2，变量3，变量名...
	2、作用：
		用于删除变量，同时解除与对象的关联，如果可能则释放对象
	3、自动化内存管理的引用次数:
		每个对象记录被变量绑定(引用)的数量，当为0时被销毁
身份运算符
	语法：
		x is y
		x is not y
	作用：
		is用于判断两个对象是否是同一个对象，如果是返回True,否则返回False
		is not用于判断两个对象是否不是同一个对象，如果不是同一个对象返回True,否则返回Fase运算符优先级
	高到低：
		算数运算符：
		比较运算符
		快捷运算符
		身份运算符
		逻辑运算符
行
	1、物理行：程序员编写的行。
	2、逻辑行：Python解释器需要执行的指令。
	3、建议一个逻辑行在一个物理行上
	4、如果一个物理行中使用多个逻辑行，需要使用分好;隔开
	5、如果逻辑航过长，可以使用隐式换行或者显示换行。
		隐式换行：所有括号的内容换行，称为隐式换行
			括号包括：()	[]	{}三种
		显示换行：通过折行符\(反斜杠)换行，必须放在一行的末尾，目的是告诉解释器下一行也		    是本行的语句。
pass 语句
	通常用来填充语法空白
选择语句
if  elif  else 语句
	1、作用：
		让程序根据条件选择性的执行语句。
	2、用法：
		if 条件1:
			语句块1
		elif 条件2:
			语句块2
		else:
			语句块3
	3、说明
		elif 子句可以有0个或者多个
		else 子句可以有0个或者1个，且只能放在if语句的最后
if的真值表达式
	if 100:
		print("真值")
	等同于
	if bool(100):
		print(“真值”)
条件表达式
	语法：变量 = 结果1 if 条件 else 结果2
	作用：根据条件（True/False）来决定返回结果1还是结果2
循环语句
while 语句
	1、作用
		可以让一段代码满足条件，重复执行
	2、语法
		while 条件:
			满足条件执行的语句
		else:
			不满足条件执行的语句
	3、说明
		else字句可以省略
		在循环体内用break终止循环时，else子句不执行
for  语句
	1、作用
		用来遍历可迭代对象的数据元素
		可迭代对象是指能一次获取数据元素的对象，例如：容器
	2、语法
		for 变量列表 in 可迭代对象:
			语句块1
		else:
			语句块2
	3、说明
		else字句可以省略
		在循环体内用break终止时，else字句不执行
range  函数
	1、作用
		用来创建一个生成一系列整数的可迭代对象(也叫正数生成器)
	2、语法
		range(开始,结束,步长)
	3、说明
		函数返回的可迭代对象可以用for取出其中元素
		返回的数字不包含结束点
		开始点默认为0
		间隔默认为1
